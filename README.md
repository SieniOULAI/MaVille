# MaVille - PWA Civique

Une Progressive Web App (PWA) pour la participation citoyenne en Côte d'Ivoire avec design amélioré et fonctionnalités avancées.

## Fonctionnalités

- **Accueil** : Flux d'actualités locales avec animations et cartes interactives
- **Avis** : Sondages citoyens anonymes avec stockage local des votes
- **Signaler** : Signalements rapides avec géolocalisation automatique
- **Proximité** : Annuaire des services locaux
- **Moi** : Espace personnel avec statistiques
- **Thème sombre/clair** : Toggle avec sauvegarde des préférences
- **Navigation tactile** : Swipe pour naviguer entre sections
- **Animations fluides** : Transitions et effets visuels
- **Mise en cache avancée** : Fonctionnement hors ligne
- **Stockage local** : Signalements et votes sauvegardés localement

## Technologies

- HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- PWA APIs (Service Worker, Manifest, Geolocation, Notifications)
- Stockage local (localStorage)
- Animations CSS et JavaScript

## Déploiement sur Netlify

1. Pousser le code sur un repository Git
2. Connecter à Netlify pour déploiement automatique
3. La PWA s'installe automatiquement sur les appareils mobiles
